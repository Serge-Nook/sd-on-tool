# SD-ON Tool

Автономное приложение для **Steam Deck** — радио, темы оформления и управление
насыщенностью экрана. Работает офлайн (кроме потокового радио), **не требует**
Decky Loader, магазинов плагинов и загрузки контента из сети.

Возможности:

- **Радио** — 8 встроенных станций + пользовательские (add/edit/delete),
  воспроизведение через HTML5 `<audio>`, независимая громкость, автопереподключение.
- **Темы** — 20 встроенных тем оформления приложения.
- **Цвета экрана** — регулятор насыщенности 100–200%.
- **О программе** — версия, разработчик, сайт.

> **Важно про интеграцию в Steam.** Встроить в SD-ON Tool **отдельную вкладку в
> Quick Access Menu** без Decky/инъекции в CEF технически невозможно: интерфейс
> Steam — React-приложение, а правка `steamui/index.html` не исполняется на
> актуальном SteamOS. Поэтому SD-ON Tool сделан как **самостоятельное приложение**
> (отдельное окно). Его можно добавить в библиотеку как стороннюю игру
> («Add a Non-Steam Game») и запускать прямо в игровом режиме.

## Структура репозитория

```
app/                          — автономный веб-интерфейс приложения
  index.html                  — точка входа
  app.js                      — вся бизнес-логика (radio/themes/display/about)
  app.css                     — базовые стили (переменные --sdon-*)
  themes/                     — 20 CSS-тем
  assets/icons, assets/fonts  — ресурсы
bin/sd-on-tool                — лаунчер (открывает app в окне браузера --app)
packaging/sd-on-tool.desktop  — шаблон ярлыка меню приложений
install.sh                    — GUI-диспетчер (zenity/kdialog, fallback — CLI)
scripts/
  installer.sh                — установка/удаление
  gen-themes.sh               — генератор 20 тем
  build-sh.sh                 — сборка самодостаточного SD-ON-Tool-Installer.sh
```

## Установка

Скачайте из релиза `SD-ON-Tool-Installer.sh`, сделайте исполняемым и запустите —
откроется окно с кнопками **«Установить»** / **«Удалить»**:

```bash
chmod +x SD-ON-Tool-Installer.sh
./SD-ON-Tool-Installer.sh
```

Это самодостаточный файл: всё приложение (app, темы, лаунчер) встроено внутрь и
распаковывается при запуске.

Из клона репозитория установщик можно запустить напрямую:

```bash
./install.sh                       # графический диалог или CLI
bash scripts/installer.sh install  # без GUI
```

Собрать `.sh`-пакет самостоятельно:

```bash
bash scripts/build-sh.sh   # результат: dist/SD-ON-Tool-Installer.sh
```

Установщик:

1. Копирует приложение в `~/.local/share/sd-on-tool/`.
2. Создаёт лаунчер `~/.local/bin/sd-on-tool` и ярлык в меню приложений.
3. **Файлы Steam не изменяются**, права root не нужны, фоновых сервисов нет.

Запуск: из меню приложений («SD-ON Tool») или командой `~/.local/bin/sd-on-tool`.
Для игрового режима добавьте лаунчер как стороннюю игру (Steam → Add a Non-Steam Game).

## Удаление

Кнопка **«Удалить»** в установщике или:

```bash
bash scripts/installer.sh uninstall
```

Удаляются лаунчер, ярлык и директория `~/.local/share/sd-on-tool/`.

## Требования

Нужен установленный браузер на базе Chromium (Chromium / Google Chrome / Brave /
Edge — в т.ч. Flatpak) для окна-приложения; при их отсутствии используется
Firefox или системный `xdg-open`. На Steam Deck браузер ставится из Discover.

## Хранение настроек

Настройки (тема, насыщенность, станции, громкость) хранятся в `localStorage`
приложения. Файлы Steam и система не затрагиваются.

## Ограничения

- Вкладки внутри Quick Access Menu нет (см. врезку выше) — приложение открывается
  отдельным окном.
- Насыщенность применяется к окну приложения (CSS `saturate`), а не ко всему
  экрану: общесистемная регулировка требует отдельного системного компонента
  (`xrandr`/DRM), которого в автономном приложении нет.
